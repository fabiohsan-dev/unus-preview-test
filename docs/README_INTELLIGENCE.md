# UNUS — Hub de Inteligência Imobiliária & Data-Driven Ecosystem

## 1. Visão Geral
Este repositório representa a convergência tecnológica entre **performance (Next.js)** e **autoridade de conteúdo (WordPress)**. O projeto UNUS não é apenas uma plataforma de anúncios, mas um ecossistema projetado para transformar dados brutos do mercado imobiliário de alto padrão em decisões estratégicas para o cliente final.

---

## 2. Análise do Status Atual (Audit)

### 🏗️ Arquitetura Técnica
*   **Core Frontend (Next.js):** Altíssima maturidade. Utiliza SSR/ISR para performance extrema e SEO.
*   **Content Engine (WordPress):** Implementação de tema customizado (`unus-blog-theme`) concluída, garantindo 100% de consistência visual com o frontend.
*   **Data Source (Vista API):** Integração ativa. O fluxo de dados de imóveis é a espinha dorsal do sistema.

### 🎨 Design System
*   **Maturidade Visual:** Nível Editorial de Luxo.
*   **Tokens:** Implementação robusta de variáveis CSS (Cormorant Garamond, Inter, Grid de 8px).
*   **UX:** Foco em minimalismo e "respiro" visual, elevando a percepção de valor dos ativos listados.

## 4. Ativos de Inteligência Identificados

1.  **Dados de Inventário (Vista API):** Histórico de preços, tipologias mais buscadas e velocidade de vendas por bairro.
2.  **Dados de Infraestrutura Local:** Conteúdo sobre obras públicas (Beira-Rio, Orla, Koesa) que impactam diretamente o valor do m².
3.  **Comportamento do Usuário:** Favoritos e filtros de busca (indicadores de demanda represada).

---

## 5. Dicas e Análise para Melhorias (Roadmap de Dados)

### A. Evolução do Conteúdo (Blog → Intelligence)
*   **Mudar o "O que" pelo "Por que":** Parar de listar "vantagens de morar no litoral" e começar a publicar "Índice de Valorização do Litoral Catarinense vs. IPCA".
*   **Hyper-Local Insights:** Criar uma página de dados para cada bairro (ex: "Bairro Campinas: Raio-X do m² e projeção de novos lançamentos").

### B. Funcionalidades de Dados (Next.js)
*   **Calculadora de ROI:** Para imóveis de investimento, permitir que o usuário calcule o retorno de aluguel/revenda baseado nos dados reais da Vista API.
*   **Comparador de Ativos:** Permitir comparação lado a lado de métricas técnicas (valor do condomínio/m², insolação, vagas de garagem).

### C. Melhorias de UX
*   **Filtros Inteligentes:** Implementar busca por "Lifestyle" (ex: "Próximo a marinas", "Vista mar definitiva", "Walkability alto").
*   **Social Proof Real-Time:** "3 pessoas viram este imóvel nas últimas 24h" (Gera escassez para alto padrão).

### D. SEO de Autoridade
*   **Snippets de Dados:** Configurar o WordPress para enviar dados estruturados (Schema.org) de "RealEstateListing" e "AnalysisScore" para o Google.
